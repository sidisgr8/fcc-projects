Introduction

Python is a high-level, interpreted programming language that emphasizes code readability with the use of significant indentation.

Created by Guido van Rossum and released in 1991, Python's design philosophy emphasizes code readability with the use of significant whitespace.

Python features a dynamic type system and automatic memory management and supports multiple programming paradigms.

  • Easy to learn and read
  • Large standard library
  • Active community support
Getting Started

To begin with Python, you'll need to download and install it from python.org.

After installation, you can verify Python is installed by opening a terminal and typing:

python --version

Create your first Python program:

print("Hello, World!")
Data Types

Python has several built-in data types that you'll use frequently.

Here are the most common ones:

  • Numbers (int, float)
  • Strings (str)
  • Lists (list)
x = 42 # Integer y = 3.14 # Float name = "Python" # String nums = [1, 2, 3] # List
Control Flow

Control flow lets you control which instructions are executed based on conditions.

Python uses indentation to define blocks of code.

if condition: # do something elif other_condition: # do something else else: # do another thing
  • if statements for conditional execution
  • for loops for iteration
  • while loops for conditional iteration
Functions

Functions are blocks of reusable code that perform specific tasks.

Define functions using the def keyword:

def greet(name): return f"Hello, {name}!" # Call the function message = greet("Python")

Functions can have parameters and return values.

They help organize code and prevent repetition.
